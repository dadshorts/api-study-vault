# API-Foundations Study Vault — Index

8-week curriculum to learn API infrastructure end-to-end: types → build → break.
JS-first, Node + Express stack. Two deliverable artifacts: portfolio API + crAPI pentest report.

## How to Use This Vault

1. Read `CURRICULUM.md` — the 8-week plan, source of truth
2. Complete `prereq-checklist.md` (Week 0, ~half a day)
3. Work phase-by-phase: 00 → 01 → 02 → 03
4. Each phase folder's `README.md` lists its sources, hours, and deliverables

---

## Folder Map

### Top-level
- `CURRICULUM.md` — Full 8-week plan, weighting (10/45/45), and recovery rules
- `prereq-checklist.md` — Week 0 install + setup checklist

### 00-types-orientation/ (Week 1, ~8 hrs, 10%)
Mental map of REST, GraphQL, gRPC, WebSocket, SOAP, webhooks.
**Milestone:** one-page README in your own metaphors.

### 01-build/ (Weeks 2-4, ~36 hrs, 45%)
Ship a deployed Express REST API + earn freeCodeCamp Back-End certificate.
**Artifact 1:** portfolio API on GitHub with OpenAPI spec.

### 02-break/ (Weeks 5-7, ~36 hrs, 45%)
Attack OWASP crAPI under structured guidance. Then attack your own portfolio API.
**Milestone:** PortSwigger API path solved + 3 vulns found in your own code.

### 03-capstone/ (Week 8, ~10 hrs)
Polish both artifacts to portfolio quality.
**Artifact 2:** crAPI pentest report, written like a real client deliverable.
