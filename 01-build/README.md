# Phase 1 — Build

**Weeks 2-4 · ~36 hours · 45% of total curriculum**

## Goal
Ship a deployed Express REST API with auth, persistence, and an OpenAPI spec. Earn the freeCodeCamp Back-End and APIs certificate along the way.

## Weekly breakdown

### Week 2 — Foundations (~12 hrs)
- [microsoft/Web-Dev-For-Beginners](https://github.com/microsoft/Web-Dev-For-Beginners) — Node/Express lessons (~6 hrs)
- [MDN — Express/Node Server-Side Tutorial](https://developer.mozilla.org/en-US/docs/Learn/Server-side/Express_Nodejs) — read through "Routes and controllers" (~6 hrs)

### Week 3 — fCC Cert: Projects 1-3 (~12 hrs)
[freeCodeCamp — Back-End Development and APIs](https://www.freecodecamp.org/learn/back-end-development-and-apis/)
- Timestamp Microservice
- Header Parser Microservice
- URL Shortener Microservice

### Week 4 — fCC Cert: Projects 4-5 + Portfolio API (~12 hrs)
- Exercise Tracker
- File Metadata Microservice
- **Earn the freeCodeCamp certificate**
- Start your **portfolio API** (e.g., a "BookShelf" CRUD service): users, JWT auth, Postgres or MongoDB

## Deliverables (commit to this folder)

- [ ] `fcc-projects/` — links to all 5 fCC project repos
- [ ] `fcc-cert-url.md` — paste your freeCodeCamp certificate URL here
- [ ] `portfolio-api/` — your real API project (or link out to its own repo)
- [ ] `security-self-grade.md` — score against [shieldfy/API-Security-Checklist](https://github.com/shieldfy/API-Security-Checklist)

## Verification

- [ ] freeCodeCamp Back-End cert URL is committed
- [ ] Portfolio API deployed (Render, Railway, or Fly.io free tier) — URL committed
- [ ] OpenAPI 3 spec exposed at `/openapi.json`
- [ ] `npm test` passes with at least 5 integration tests
- [ ] Self-graded security checklist score ≥ 60%

## Why this phase is the longest *paid* phase

You can read about APIs forever. Until you've shipped one — survived auth bugs, deployed it, written a real OpenAPI spec, and seen requests fail in production — you don't actually know APIs. The fCC projects are warm-ups; the portfolio API is the real exercise.

This is also the artifact recruiters can click and inspect. Make it production-shaped, not toy-shaped.
